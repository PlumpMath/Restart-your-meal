# Game-Off-2017

Game Off 2017 game


## Ideas

### general

- 3D game

### gameplay

- redo same level
- character has to avoid to meet previous ghosts
- respawn in different point at each run
- character has to collect max pickable objects
- timer to set difficulty level easily
